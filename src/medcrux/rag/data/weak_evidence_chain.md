# 弱证据链知识库

> **文档性质**: 弱证据链知识（Weak Evidence Chain）
> **维护者**: MedCrux MD（医学领域专家）
> **创建日期**: 2026-01-04
> **使用原则**: 本知识库包含未经充分验证的医学知识，证据链较弱，仅供参考

---

## 一、弱证据链知识库说明

### 1.1 建立目的
根据专家委员会要求，公理系统（`breast_ultrasound_report_axioms.md`）应该是：
- **高度浓缩的**：只包含核心、必要的公理
- **循证支持的**：每个公理都有明确的证据来源
- **逻辑自洽的**：公理之间逻辑一致，不矛盾
- **不经常改动的**：除非有明确的证据链，否则不应修改

对于**没有充分证据支持**的医学知识，应放入本弱证据链知识库，而不是公理系统。

### 1.2 使用原则

#### 1.2.1 检索层级
系统应按照以下层级检索和使用知识：
1. **公理系统**（最高优先级）：高度浓缩、循证支持、逻辑自洽
2. **弱证据链知识库**（次优先级）：未经充分验证，但可能有参考价值
3. **其他来源**（最低优先级）：网络搜索、临时知识等

#### 1.2.2 建议标注
当MedCrux给出建议时，必须明确标注建议的证据链：
- **公理系统**：标注为"基于ACR BI-RADS标准"或"基于公理X"
- **弱证据链**：标注为"基于弱证据链，仅供参考"
- **其他来源**：标注为"基于其他来源，需进一步验证"

### 1.3 证据链强度分级

#### 证据链强度1：强证据（应放入公理系统）
- **标准**：ACR BI-RADS v2025、中华医学会超声医学分会2025等权威标准
- **特征**：有明确的官方发布、权威机构认可、广泛采用
- **使用**：直接用于逻辑一致性检查、风险评估等核心功能

#### 证据链强度2：中等证据（可放入弱证据链）
- **标准**：医学文献、专家共识、临床经验总结
- **特征**：有文献支持，但未纳入权威标准，或证据不够充分
- **使用**：作为参考信息，需明确标注"弱证据链"

#### 证据链强度3：弱证据（可放入弱证据链）
- **标准**：个案报告、初步研究、网络信息
- **特征**：证据不足，但可能有参考价值
- **使用**：仅作为补充信息，必须明确标注"弱证据链，仅供参考"

## 二、弱证据链知识条目

### 2.1 形态学特征相关

#### 条目1：条索状/条状低回声与导管型乳腺癌的关系
- **证据链强度**：弱证据（证据链强度3）
- **证据来源**：
  - 网络搜索未找到明确的权威标准支持
  - 部分医学文献提到可能相关，但证据不足
  - 临床经验中有相关案例，但未纳入BI-RADS标准
- **内容**：
  - 条索状/条状低回声可能与导管型乳腺癌相关
  - 但未纳入ACR BI-RADS标准作为标准恶性征象
  - 需要结合其他征象（如导管内占位、血流信号、微钙化）综合判断
- **使用建议**：
  - 不应作为独立的恶性征象判断标准
  - 可作为补充信息，需明确标注"弱证据链，仅供参考"
  - 建议结合其他征象综合评估

#### 条目2：其他非标准形态学描述
- **证据链强度**：弱证据（证据链强度3）
- **内容**：
  - 各种非标准术语（如"管状"、"线状"等）的医学意义
  - 这些术语未纳入ACR BI-RADS标准术语集合
  - 可能在不同文献或临床实践中有不同的解释
- **使用建议**：
  - 不应作为标准判断依据
  - 可作为参考信息，需明确标注"弱证据链"

### 2.2 征象组合相关

#### 条目3：特定征象组合的恶性可能性
- **证据链强度**：中等证据（证据链强度2）
- **证据来源**：
  - 部分医学文献提到特定征象组合可能提示恶性
  - 但未纳入ACR BI-RADS标准作为标准判断依据
- **内容**：
  - 某些征象组合（如"条索状低回声 + 导管内占位 + 血流信号 + 微钙化"）可能高度提示恶性
  - 但这些组合未纳入标准BI-RADS分类依据
- **使用建议**：
  - 可作为参考信息，需明确标注"基于弱证据链，仅供参考"
  - 不应替代标准BI-RADS分类判断

## 三、弱证据链知识的使用规范

### 3.1 检索优先级
1. **首先检索公理系统**：使用公理系统进行核心判断
2. **其次检索弱证据链**：如果公理系统无法覆盖，再检索弱证据链
3. **最后检索其他来源**：如果两者都无法覆盖，再检索其他来源

### 3.2 建议标注规范
当使用弱证据链知识时，必须在建议中明确标注：

```
基于弱证据链，仅供参考：
- [具体内容]
- 证据来源：[来源说明]
- 建议：[建议内容]
```

### 3.3 风险评估
- **公理系统**：直接用于风险评估（Low/Medium/High）
- **弱证据链**：仅作为补充信息，不应直接用于风险评估
- **其他来源**：不应用于风险评估

## 四、弱证据链知识的维护

### 4.1 添加新条目
- **条件**：有初步证据，但不足以纳入公理系统
- **流程**：
  1. MD审查证据来源和强度
  2. 确定证据链强度等级
  3. 添加到本知识库
  4. 标注证据来源和强度

### 4.2 升级到公理系统
- **条件**：弱证据链知识获得充分证据支持
- **流程**：
  1. MD审查证据是否充分
  2. 确认符合公理系统要求（高度浓缩、循证支持、逻辑自洽）
  3. 从弱证据链知识库移除
  4. 添加到公理系统
  5. 更新版本号

### 4.3 删除条目
- **条件**：证据被证伪，或不再有参考价值
- **流程**：
  1. MD审查并确认
  2. 从弱证据链知识库移除
  3. 记录删除原因
